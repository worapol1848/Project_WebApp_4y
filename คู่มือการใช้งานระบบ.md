# 📘 คู่มือการใช้งานระบบ Velin Inventory & E-Commerce

คู่มือฉบับนี้จัดทำขึ้นอธิบาย **"วิธีการใช้งาน" (User Manual)** แบ่งแยกระหว่าง **ผู้ใช้งานทั่วไป (Customer)** และ **ผู้ดูแลระบบ (Admin)** อย่างชัดเจน

---

## 🚀 1. การเปิดระบบ (Starting the System)
การเข้าทดสอบระบบสามารถทำได้ทั้งการ "คลิก" และการ "พิมพ์เปิด" (Terminal / Command prompt) ตามความถนัดดังนี้:

**วิธีที่ 1: รันด้วยการดับเบิลคลิก (แนะนำที่สุด)**
1. ไปที่โฟลเดอร์หลักของโปรเจกต์ `Velin Web App`
2. ดับเบิลคลิกเปิดไฟล์ **`start_dual_sessions.bat`** (เพื่อรันแยกจอ Admin กับ จอลูกค้า)
3. ระบบจะเปิดหน้าต่าง Command Line ขึ้นมาพร้อมกัน 3 หน้าต่างอัตโนมัติ 
   *(ครอบคลุมทั้ง Backend, จอ Admin: 5173 และ จอลูกค้า: 5174)*

**วิธีที่ 2: รันด้วยการพิมพ์คำสั่งผ่าน Terminal / CMD (Manual Execution)**
สำหรับคนที่ถนัดใช้ระบบ Command Line พิมพ์เอา สามารถเปิด Terminal ทีละหน้าต่างได้:
1. **ในส่วนของ Backend:** 
   ให้พิมพ์คำสั่งเพื่อเปิดการทำงานเซิร์ฟเวอร์
   ```bash
   cd backend
   npm run dev
   ```
2. **ในส่วนของ Frontend:**
   เปิด Terminal ใหม่แล้วสั่งพิมพ์เพื่อเปิดหน้า UI
   ```bash
   cd frontend
   npm run dev
   ```
   *(คุณสามารถพิมพ์คำสั่งซ้ำนี้บน Terminal อีกตัว เพื่อแยกจอทดสอบระหว่างลูกค้ากับแอดมินได้เลย)*

## 🧑‍💻 2. คู่มือสำหรับผู้ใช้งานทั่วไป (Customer Portal)

**⚠️ ข้อจำกัดสำคัญของลูกค้าระบบนี้:**
- ❌ ลูกค้า **ไม่สามารถ** เข้าสู่ระบบด้วยใบหน้าได้ (ใช้ได้แค่อีเมลและรหัสผ่านเท่านั้น)
- ❌ ลูกค้า **ไม่สามารถ** สั่งพิมพ์ไฟล์ใบเสร็จสั่งซื้อ (PDF) ได้ (สิทธิ์การปริ้นท์เอกสารสงวนให้แอดมินพิมพ์แปะกล่องพัสดุเท่านั้น)
- ❌ ลูกค้า **ไม่สามารถ** เข้าสู่หน้าหลังบ้านได้เด็ดขาด

### 🛒 2.1 ค้นหาและดูสินค้า (Product Catalog)
- **เมนูหน้าหลัก:** คุณจะเห็นสินค้าแนะนำ และหมวดหมู่
- **ตัวกรองแบรนด์ (Brand Filter):** สามารถคลิกไอคอนแบรนด์ด้านบน (เช่น Nike, Adidas) เพื่อค้นหากรองเฉพาะยี่ห้อที่ต้องการ

### 🛍️ 2.2 การเพิ่มคิวและชำระเงิน (Cart & Payment)
- ผู้ใช้ต้อง **ล็อกอินเข้าสู่ระบบ** ก่อนถึงจะกดเพิ่มสินค้าลงตะกร้าได้ (เพื่อป้องกันข้อมูลสูญหาย)
- เมื่อกด **Add to Cart** คุณต้องเลือก "ขนาด (Size)" เช่น 42 และเลือก "จำนวน (Quantity)" ข้อมูลจะถูกจับโยนลงในตะกร้ามุมขวาบน
- **หน้าชำระเงิน (Checkout):**
  1. เลือกการจัดส่ง: ระบบจะคำนวณราคาให้ (Messenger 60 บาท / EMS 150 บาท)
  2. เลื่อนปักหมุดบนแผนที่ (Map Locator) เพื่อชี้เป้าบ้าน
  3. อัปโหลดรูปภาพ "สลิปโอนเงิน" แล้วกดยืนยันการสั่งซื้อ 

### 📦 2.3 ติดตามสถานะและขอเงินคืน (My Orders)
- **ติดตามสถานะ:** ลูกค้าสามารถตรวจสอบได้ว่าของอยู่สถานะไหน (รอดำเนินการ -> กำลังจัดส่ง -> ถึงปลายทางแล้ว)
- **รับสินค้า:** เมื่อพัสดุมาส่งถึงบ้าน ต้องกดปุ่ม **"ฉันได้รับสินค้าแล้ว"** 
- **ขอเงินคืน (กรณีโดนเท):** หากแอดมินแจ้งว่าสินค้าหมดและกดยกเลิกบิล จะมีฟอร์มสีแดงเด้งขึ้นมาในหน้านี้ให้คุณ **กรอกชื่อ-เลขบัญชีธนาคาร** และกดยืนยันส่งกลับไปขอเงินคืน

## 💼 3. คู่มือสำหรับผู้ดูแลระบบ (Admin Portal)

### 📊 3.1 ภาพรวมรายได้ (Dashboard)
- แอดมินเข้าด้วยชื่อผู้ใช้ทั่วไป จะเจอกับกราฟยอดขาย ยอดออเดอร์ สรุปสินค้าฮิต และเตือนสินค้าใกล้หมดสต็อก

### 📋 3.2 กองควบคุมคิวการสั่งซื้อ (Order Management)
นี่คืองานหลักของ Admin เมื่อมี Notification ออเดอร์ใหม่เข้ามา:
1. **ตรวจสอบสลิป:** กดดูรูปสลิปว่ายอดเงินตรงกับราคารวมไหม ถ้าตรงให้กด **ยืนยันการชำระเงิน**
2. **พิมพ์ใบเสร็จ (PDF Print):** แอดมินจะกดยืนยันการดาวน์โหลดหรือสั่งปริ้นท์ใบเสร็จที่ถูกรวบรวมออกมาเป็น PDF นำไปติดกับกล่องพัสดุ
3. **จัดส่งสินค้า:** เมื่อส่งของที่ขนส่งเสร็จ แอดมินมากดเปลี่ยนสถานะออเดอร์เป็น **กำลังจัดส่ง (Shipped)** หรือกดให้เป็น **ถึงปลายทางแล้ว (Arrived)**

### 📦 3.3 การยกเลิกสินค้าและการควบคุมคลัง (Refund & Inventory)
- **วิกฤตสินค้า (Cancel):** หากหาของไม่เจอ แอดมินกดยกเลิกสินค้า พร้อมพิมพ์เหตุผล (ระบบจะคืนสต็อกพ่วงให้อัตโนมัติทันที) จากนั้นรอให้ลูกค้ากรอกเลขบัญชีโอนคืนมาให้
- **การคืนเงิน:** เมื่อโอนเงินแล้ว ให้เข้ามากดปุ่มยืนยันปิดคดีถือว่า **Refunded Complete**
- **เพิ่มสินค้าใหม่:** ในหน้า Inventory ของแอดมิน สามารถใส่ภาพ แยกไซส์ และกำหนดแบรนด์ได้

## 👑 4. คู่มือสำหรับซูเปอร์แอดมิน (Super Admin Portal)
การทำงานระดับขั้นสูงที่มีจุดมุ่งหมายด้าน Security มีเฉพาะขั้นสูงสุดที่ทำได้:

### 🔒 4.1 เข้าสู่ระบบด้วยใบหน้า (Face-Scan Biometric)
- เพื่อเข้าถึงหน้า Super Admin คุณต้องกดเลือกการเข้าสู่รูปแบบ Face-Scan ระหว่างล็อกอิน
- กล้องเว็บแคมเซิร์ฟเวอร์จะถูกเปิดขึ้นมาเทียบเค้าโครงหน้ากับฐานข้อมูล เมื่อตรวจผ่านพิกัดถึงจะเข้าทำงานระบบหลังบ้านได้

### 🕵️ 4.2 ตรวจจับทุจริตลูกจ้างพนักงาน (System Audit Logs & Admin Manager)
- **ควบคุมพนักงาน:** สิทธิ์ในการอนุมัติใครเป็นแอดมินใหม่ ลบแอดมินเก่า หรือรีเซ็ตภาพใบหน้าผ่านการสแกนใหม่
- **System Logs:** ทะเบียนประวัติความเคลื่อนไหว ที่บันทึกไว้ว่า แอดมิน A ดำเนินการลบสินค้า แอดมิน B ยกเลิกยอดเงินคืนเมื่อเวลา X นาฬิกา (ตรวจสอบย้อนหลังได้ทุกการกระทำ)
